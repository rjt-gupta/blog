This open-to-all program is for students who want to get started with opensource world and to learn about how to contribute on Github! ![Smithsonian Image]({{ site.url }}/images/koss.PNG) {: .image-center} The actual benefit of participating in such programs is you get to know different people, you make connections a lot and ofcourse you learn a lot. I found out about KWoC through a friend. My major motive was to get aquainted with git and github (even though I have done some personal projects) for GSoC. Through this I got to know many things about Open Source Culture. ## Project I am passionate about Information Security so I selected **VAULT** from a list of more than 100 projects which is a **Swiss Army Knife for Hackers**. I communicated with my mentor *Abhishek Sharma* and started contributing. ## My Contributions: ### **Pull Request #1:** So I was setting up the development environment and just noticed that the project was not ignore the files generated by Pycharm IDE and guess what I just made my first contribution by adding .idea and /venv to .gitignore file. ![Smithsonian Image]({{ site.url }}/images/kwoc-pr1.png) {: .image-center} ### **Pull Request #2:** I was going through the issues and selected issue **#29**. In this I had to implement a validation to check wether the inputted ip is valid or not. So I used regular expression and created a small check function for ip addresses. ![Smithsonian Image]({{ site.url }}/images/kwoc-pr2.png) {: .image-center} ### **Pull Request #3** After my second PR got merged I started working on implementing an output functionality for each and every module. First I made output function for SSL module. ![Smithsonian Image]({{ site.url }}/images/kwoc-pr3.png) {: .image-center} ### **Pull Request #4** Next I implemented text output function for the Information Gathering module. This PR was interesting. ![Smithsonian Image]({{ site.url }}/images/kwoc-pr4.png) {: .image-center} ### **Pull Request #5** Then came the turn for the comments module, I made an easy output functionality. ![Smithsonian Image]({{ site.url }}/images/kwoc-pr5.png) {: .image-center} ### **Pull Request #6** In this Pull Request I fixed the Google Dorking module for some invoking and PEP-8 issues and created output function for whois module. ![Smithsonian Image]({{ site.url }}/images/kwoc-pr6.png) {: .image-center} So Yeah! this has been an amazing experience so far and I am looking forward to continue after the mid evaluations.
